CMU Tower Defense is a clone of the famous Bloons Tower Defense 6 game, by NinjaKiwi, made for the 15-112 Fundamentals of Programming and Computer Science. In this game you have to place monkeys in strategic locations on a map in order to pop balloons before they get to the end of the map. Balloons have different speeds and properties. When you pop a higher level balloon, it generates a balloon from the previous level. The game has different monkeys and towers to choose from.

How to play?

- pip install -r requirements.txt
- python cmu_tower_defense/main.py
